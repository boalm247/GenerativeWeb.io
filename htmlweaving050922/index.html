About

This project is an exploration between the connection of computer language and textile weaving. Early mechanical textile looms, specifically the Jaquard Loom, worked with interchangable punchcards, which later inspired the first computers. More often our interactions with computers dictate a desired output. This project reverses the role, by generating outputs dictated by the computer.

Anyone creating a weaving from the Weaving Pattern Generator will have all the parameters set for them by the computer. The patterns work best for the Schacht Loom. These looms are available in the Textile Lab at Parsons School of design.
